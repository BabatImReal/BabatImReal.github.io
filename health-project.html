Health Data Analysis Project

Project Overview

Comprehensive end-to-end analysis of a year's worth of personal health data to uncover trends, correlations, and actionable behavioral insights for improved wellness.

Data Extraction

Extracted comprehensive health metrics including sleep duration, step count, walking speed, and active energy from Apple Health using Python-based data crawling techniques.

Data Transformation

Cleaned and transformed raw health data with Python and Power Query, preparing structured datasets for analytical processing and visualization.

Interactive Dashboards

Visualized key insights using Power BI, building interactive dashboards to explore activity patterns and health indicators with dynamic filtering.

Correlation Analysis

Analyzed correlations between variables (e.g., walking asymmetry vs. step count) to uncover patterns influencing physical performance and overall health.

Comprehensive Reporting

Delivered a comprehensive Power BI report and summary PDF highlighting major findings and data-driven recommendations for wellness improvement.

Technologies Used

Python

Power BI

Power Query

Analytics Dashboard

Key Insights & Findings

Activity Patterns

Daily step count analysis reveals peak activity periods and identifies opportunities for increased movement throughout the day.

Sleep Quality

Sleep duration trends show correlations with daily activity levels and provide insights into optimal rest patterns for better recovery.

Health Metrics

Walking speed and energy expenditure metrics highlight physical performance trends and effectiveness of fitness routines over time.

Behavioral Trends

Temporal analysis reveals weekly and monthly patterns, enabling data-driven adjustments to lifestyle and fitness goals.
